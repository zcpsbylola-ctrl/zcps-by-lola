# ZCPS by Lola — Système de Réservation Luxe

Ce projet est une application web mobile-first de réservation pour **ZCPS by Lola**.

## Caractéristiques
- **Interface Luxe** : Design épuré avec des dégradés et une typographie soignée.
- **Mobile-First** : Optimisé pour une utilisation fluide sur smartphone.
- **Timeline 24h** : Visualisation continue des créneaux.
- **Buffer de Sécurité** : Blocage automatique de 75 minutes après chaque réservation.
- **Intégration WhatsApp** : Validation directe via un message pré-rempli.

## Déploiement
Le projet est hébergé sur GitHub Pages.

## Utilisation
1. Ouvrez l'application.
2. Faites défiler horizontalement pour choisir le jour.
3. Sélectionnez un créneau horaire.
4. Cliquez sur **Valider** pour envoyer la demande via WhatsApp.
